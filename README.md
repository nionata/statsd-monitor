# statsd monitor

Monitor the metrics being published with the statsd protocol.

## Contributing

### Setup

1. Install nix https://github.com/DeterminateSystems/nix-installer
2. Install direnv

```bash
$ nix profile install nixpkgs#direnv
$ nix profile install nixpkgs#nix-direnv
```

3. Open a dev shell

```bash
$ nix develop
```
